# 📸 Imágenes de Memes

## Especificaciones

- **Dimensiones**: 256x256 px (cuadradas)
- **Formato**: PNG
- **Tamaño**: < 100KB cada una
- **Nombres**: 1.png, 2.png, 3.png, ... hasta 42.png

## Ubicación

Coloca las imágenes en esta carpeta:
```
public/memes/
```

## Ejemplo

```
public/memes/
├── 1.jpg
├── 2.jpg
├── 3.jpg
...
└── 42.jpg
```

## Notas

- Cada botón del juego mostrará una imagen aleatoria
- Las imágenes se mezclan en cada nivel
- Puedes cambiar las imágenes sin modificar el código
